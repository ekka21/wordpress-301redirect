A Simple 301 redirect plugin that is really easy to manage by using CSV Import/Export

== Description ==

Simple 301Redirect is a 301 HTTP redirect plugin that allows user to redirect their old urls to new urls permanently. There is import/export to CSV.

== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.
